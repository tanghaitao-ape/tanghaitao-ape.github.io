vim设置

Posted on 2021-03-07，1 min read

vim ~/.vimrc
功能：一个tab等于4个空格，显示行数，将每行对于的空格替换为点。

set ts=4
set softtabstop=4
set shiftwidth=4
set expandtab
set autoindent
set nu
set list
set listchars=tab:>\\,trail:·,extends:#,nbsp:.
let g:SuperTabDefaultCompletionType = "context"
let g:snips_trigger_key='<F2>' 
`
主题
地址：[https://github.com/altercation/vim-colors-solarized](https://github.com/altercation/vim-colors-solarized)
安装

cd vim-colors-solarized/colors
mv solarized.vim ~/.vim/colors/

 然后在 ~/.gvimrc 文件中添加以下几行命令

syntax enable
set background=light
colorscheme solarized
